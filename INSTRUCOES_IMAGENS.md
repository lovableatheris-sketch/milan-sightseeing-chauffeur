# Instruções para Adicionar as Imagens na Galeria

## Passo 1: Adicionar as Imagens
Adicione as 5 imagens fornecidas na pasta `src/assets/` com os seguintes nomes:

1. **mountain-view-1.jpg** - Vista de montanha nevada do interior do veículo
2. **mountain-view-2.jpg** - Estrada com edifício "da Aurelio" 
3. **mountain-view-3.jpg** - Paisagem de montanha nevada com pegadas
4. **fuso-service.jpg** - Van preta em frente ao serviço FUSO
5. **vehicle-interior-luxury.jpg** - Interior luxuoso com teto solar panorâmico

## Passo 2: Verificar os Imports
O arquivo `src/components/ExperienceGallery.tsx` já está configurado para importar essas imagens. 
Se você usar nomes diferentes, atualize os imports no início do arquivo.

## Localização
A galeria aparece na página **Chi Siamo** (`/chi-siamo`) logo após a seção principal de texto e imagem.

## Funcionalidades
- Carrossel automático com transição suave
- Navegação por botões laterais
- Indicadores de posição (dots)
- Efeito hover nas imagens
- Responsivo para mobile e desktop
- Autoplay pausa ao interagir
